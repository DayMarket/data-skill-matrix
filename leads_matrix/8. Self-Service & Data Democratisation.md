# 8. Self-Service и Демократизация Данных / Self-Service & Data Democratisation

Этот трек оценивает способность руководителя масштабировать доступ к данным и аналитике в организации. Он охватывает создание инструментов самообслуживания, развитие data literacy у бизнес-пользователей и выстраивание баланса между централизованной экспертизой и широким self-service доступом. С ростом уровня оценивается масштаб программ демократизации данных.

## M1 (Lead)
- Онбордит и обучает бизнес-пользователей работе с инструментами данных, давая им возможность уверенно отвечать на собственные вопросы.
- Создаёт и владеет self-service инструментарием для прямых стейкхолдеров (дашборды, отчёты), используя сертифицированные датасеты для измеримого сокращения ad-hoc запросов.

## M2 (Senior Lead)
- Масштабирует data literacy по всему домену через высокоэффективные программы (например, тренинги), измеряя их бизнес-эффект (ROI).
- Драйвит стратегию self-service для крупного бизнес-домена, трансформируя аналитическую команду из «фабрики отчётов» в мультипликатор силы.

## M3 (Head)
- Формирует культуру data literacy на уровне руководства, держа peer-руководителей ответственными за способность их команд самостоятельно принимать data-informed решения.
- Определяет философию демократизации данных для всего вертикала, задавая стратегию баланса между централизованной экспертизой и широким self-service доступом.
