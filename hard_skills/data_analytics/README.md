# Data Analytics

## О роли

Data Analyst — специалист, который помогает бизнесу принимать решения на основе данных. Роль сочетает техническую экспертизу с глубоким пониманием продукта и бизнес-процессов, а также умением доносить insights до стейкхолдеров.

## Ключевые зоны ответственности

- Проведение аналитических исследований и формирование рекомендаций
- Проектирование и проведение экспериментов (A/B-тесты)
- Построение метрик и отслеживание здоровья продукта
- Создание и поддержка аналитических моделей данных

## Что важно для этой роли

**Техническая база** — уверенное владение SQL, Python и инструментами анализа данных. Способность самостоятельно извлекать, обрабатывать и анализировать данные.

**Продуктовое мышление** — понимание бизнес-контекста, умение формулировать правильные вопросы и переводить данные в actionable insights.

**Экспериментальная культура** — владение методологией A/B-тестирования, статистическая грамотность, умение проектировать и интерпретировать эксперименты.

**Коммуникация и влияние** — способность ясно доносить результаты анализа, строить data storytelling и влиять на принятие решений через данные.

**Analytics Engineering** — создание надёжных, масштабируемых моделей данных и пайплайнов для самообслуживания команд.
